
cloudtb is a python module intended to unify the tools developed at cloudformdesign.com,
as well as provide TOTALLY FREE python convinience functions to anyone who wants them.

It makes use of the following modules, which require that the license be included:
- pyperclip (pyperclip.py)

All of these modules are located in the folder "external"

Every other function can be copied, reused, re-distributed, or anything else that anyone
wants with no attribution necessary. For more details see the license.
